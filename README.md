# Bem vindo!

Olá, esse projeto tem o objetivo de facilitar o envio de frases prontas em por via da criação de macros no teclado, usando uma tecla de gatilho como por exemplo o **F7** pode-se enviar uma frase pré-pronta no bate-papo do jogo, nosso jogo alvo no momento é o **[League of Legends](https://na.leagueoflegends.com/pt-br/)** mas nada nos impede de implementar essa ferramenta em outros jogos.


# Ferramentas
O programa é feito em C# em seu background e sua interface feita em Windows Forms.
